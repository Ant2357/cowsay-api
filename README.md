# cowsay-api
![cowsay API の宣伝画面](README-banner.png "cowsay API の宣伝画面")

cowsay 系統の機能を提供する API

## Usage

### cowsay
```sh
https://cowsay-api.ant-npb.workers.dev/api/cowsay?text=hoge
```
### output
```json
{
  "text": "[cowsay の出力結果(長いので省略)]"
}
```

## Development
```
npm install
npm run dev
```

```
npm run deploy
```
